## DeepSeek++ 1.11.4

1.11.4 是一次工具工作流和 Windows Firefox 本机连接稳定性更新，重点修复工具授权、工具结果续跑与多浏览器 Shell MCP 安装相互影响的问题。

### 主要变化

- **工具授权恢复**：修复新建、切换或刷新 DeepSeek 会话后，已确认工具仍可能被错误拒绝的情况；侧边栏的「授权全部网站」可以再次正常发起网站权限确认。
- **工具续跑恢复**：工具完成和会话结束事件按正确顺序处理，减少联网搜索等工具执行一次后对话不继续生成的问题。
- **MCP 工具列表稳定性**：服务健康检查和状态更新不会再错误清空已发现的工具列表。
- **Windows Firefox Shell MCP**：每个 Windows 浏览器使用独立的本机 Host 清单，Firefox 与 Chrome、Edge 或 Chromium 同时安装时不再互相覆盖。
- **权限变化**：Chrome、Edge 和 Firefox 均不新增浏览器权限。

Windows Firefox 用户在升级扩展后，需要重新安装一次 Shell Host 并重启 Firefox：

```bash
npx deepseek-pp-shell-host@1.11.4 install --browser firefox
```

### 验证

- `npm run ci:quality`
- `npm pack --dry-run --workspace packages/shell-host`
- `node scripts/web-search-smoke.mjs`

### Chrome Web Store 更新说明

中文：

- 修复切换或刷新 DeepSeek 会话后工具被错误拒绝的问题。
- 修复工具执行完成后可能不继续生成的问题，并保持 MCP 工具列表稳定。
- 修复 Windows Firefox 与其他浏览器同时安装 Shell MCP 时的本机 Host 冲突。
- Chrome 版本不新增浏览器权限。

English:

- Fixes confirmed tools being rejected after switching or reloading a DeepSeek conversation.
- Fixes cases where a conversation did not continue after a tool completed, while keeping MCP tool lists stable.
- Fixes the Windows native-host conflict when Firefox Shell MCP is installed alongside another browser.
- The Chrome package adds no new browser permissions.

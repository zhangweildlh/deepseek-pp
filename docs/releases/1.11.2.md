## DeepSeek++ 1.11.2

1.11.2 是一次稳定性更新，重点让升级后的基础聊天和既有 MCP 连接更容易恢复到正常可用状态。

### 主要变化

- **新对话恢复**：修复创建新 DeepSeek 会话时首条消息可能中断的问题，普通聊天可以正常开始并收到回复。
- **既有 MCP 连接恢复**：升级后会识别并恢复仍然有效的 MCP 服务配置，减少因旧缓存而需要手动重建连接的情况。
- **会话连续性**：页面创建或切换会话时，已授权工具继续按用户正在查看的浏览器会话工作，避免影响正常聊天。
- **权限变化**：Chrome、Edge 和 Firefox 均不新增浏览器权限。

### 验证

- `npm run ci:quality`
- `npm pack --dry-run --workspace packages/shell-host`
- `node scripts/web-search-smoke.mjs`

## DeepSeek++ 1.11.7

1.11.7 是一次长对话恢复与 MCP 连接可靠性更新，重点减少刷新、页面切换或消息重新挂载后历史工具记录消失，以及已保存 MCP 服务状态不同步的问题。

### 主要变化

- **长对话恢复更可靠**：历史工具卡片、Artifact 结果和 Agent 续跑记录会在对应消息于刷新、路由切换或分段加载后出现时恢复显示。
- **对话显示更干净**：历史记录保留用户实际输入的斜杠命令，不会把 Skill 的内部展开说明显示在对话里。
- **MCP 服务状态更一致**：服务发现结果与健康状态同步保存；连接配置在刷新期间变化时，旧结果不会覆盖最新配置。
- **权限变化**：Chrome、Edge 和 Firefox 均不新增浏览器权限。

### Chrome Web Store 更新说明

中文：

- 修复长对话刷新、页面切换或分段加载后，历史工具卡片、Artifact 结果和 Agent 续跑记录可能不显示的问题。
- 历史记录会保留用户实际输入的斜杠命令，避免显示 Skill 的内部展开说明。
- 已保存 MCP 服务的发现结果与连接状态更可靠。Chrome 版本不新增浏览器权限。

English:

- Fixes missing restored tool cards, Artifact results, and agent continuation records after a long conversation reloads, navigates, or loads in stages.
- Restored history keeps the slash command the user entered instead of showing a Skill's internal expanded instructions.
- Improves saved MCP service discovery and connection-state reliability. The Chrome package adds no new browser permissions.

## DeepSeek++ 0.7.4

0.7.4 是项目侧边栏与语言体验增强版本，让项目对话可以直接在 DeepSeek 历史侧栏中管理，并让中英文界面选择更清晰、更稳定。

### 主要变化

- **项目侧边栏进入 DeepSeek 历史区**：项目会显示在 DeepSeek 原生历史侧栏中，可展开查看相关对话，并直接打开项目内对话。
- **项目对话整理更顺手**：可以把当前对话加入项目、从项目移除，或指定下一条新会话使用某个项目上下文，减少整理任务时的来回切换。
- **原生会话菜单支持项目操作**：在 DeepSeek 历史对话菜单中可快速将对话加入项目或从项目移出，项目整理不再只依赖侧边栏页面。
- **界面语言控制更清楚**：设置页新增界面语言区域，可选择跟随浏览器、简体中文或 English，并显示当前生效语言。
- **项目入口稳定性增强**：项目侧栏渲染、菜单点击、历史标题提取和重复刷新处理更稳，降低项目入口闪烁、误隐藏或标题混入时间文本的概率。
- **回归覆盖补强**：新增项目侧栏管理、原生菜单项目操作、语言文案和历史标题提取测试，发布前继续覆盖多浏览器构建与资产校验。

English:

- Projects now appear in the DeepSeek history sidebar, where linked conversations can be expanded and opened directly.
- Move the current conversation into or out of a project, or choose which project context should apply to the next new conversation.
- DeepSeek's native conversation menu now includes project actions for faster history organization.

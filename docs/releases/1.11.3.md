## DeepSeek++ 1.11.3

1.11.3 是一次 MCP 与聊天稳定性更新，重点让已连接服务的工具列表和本地 Skill 工作流在刷新后保持可用。

### 主要变化

- **MCP 工具列表恢复**：修复服务测试成功后工具数仍显示为 0 的问题，已发现的工具列表不会因状态更新被错误清空。
- **本地 Skill 导入恢复**：Shell MCP 的本地文件夹选择工具能够继续出现在本地 Skill 导入流程中，方便选择并导入本机 Skill 文件夹。
- **聊天与工具稳定性**：减少页面运行时消息相互干扰导致的聊天或工具状态异常，让普通聊天和已授权工具连接更稳定。
- **权限变化**：Chrome、Edge 和 Firefox 均不新增浏览器权限。

### 验证

- `npm run ci:quality`
- `npm pack --dry-run --workspace packages/shell-host`
- `node scripts/web-search-smoke.mjs`

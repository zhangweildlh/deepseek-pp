## T1.5 Freeze Evidence

- Three generated MV3 manifests now have exact permission, host, identity, action/side-panel, CSP, sandbox, and web-accessible-resource assertions; Chromium/Firefox capability degradation is executable.
- DeepSeek route/body/SSE and Official API cancellation, missing-body, and `thinking_content` alias behavior are frozen; substring matching, malformed JSON, and CRLF remain owned gaps.
- MCP negotiation, all five transport identities, Native v1 emission, output envelopes, unknown transport, shallow JSON-RPC, UTF-16 truncation, and page-level tool-count overshoot are executable fixtures.
- The Shell Host is called through real four-byte framing for initialize, exact 12-tool schemas, outputs, invalid/future envelopes; installer paths/manifests cover macOS, Linux, and Windows.
- Android package/SDK, origin/bridge weakness, unsupported response, and required staging inputs have static executable checks. Asset staging passed; APK/runtime negatives remain deferred to T2.3 because JDK/Gradle is unavailable.

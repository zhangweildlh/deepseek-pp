## Retired Mobile Contract

The former `AN-001` through `AN-004` Android minimum contracts were retired by Issue [#345](https://github.com/zhu1090093659/deepseek-pp/issues/345). Their implementation, build, CI, fixtures, and current-support claims were removed; they remain visible only in historical progress and release records.

## T1.5 Freeze Evidence

- Three generated MV3 manifests now have exact permission, host, identity, action/side-panel, CSP, sandbox, and web-accessible-resource assertions; Chromium/Firefox capability degradation is executable.
- DeepSeek active/passive route/body/SSE, LF/CRLF and split UTF-8 framing, operation-specific network budgets (4 MiB active/Official API and 32 MiB per-session export), passive cancellation/backpressure, and Official API cancellation, missing-body, and `thinking_content` alias behavior are frozen; malformed JSON diagnostics remain the owned gap.
- MCP supported-version negotiation, all five transport identities, Native v1 emission, output envelopes, explicit pre-network unknown-transport rejection, shallow JSON-RPC, UTF-16 truncation, and page-level tool-count overshoot are executable fixtures.
- The Shell Host is called through real four-byte framing for initialize, exact 12-tool schemas, outputs, invalid/future envelopes; installer paths/manifests cover macOS, Linux, and Windows.
